# SV Tenacious Website

## Description
This project is a simple static website for SV Tenacious, an accessible tall ship. It provides basic information about the vessel, including an introduction, description, information on her accessible features and an image.

The website is built using HTML and CSS and is deployed using GitHub Pages.

## Features
- Clean and simple layout
- Styled using external CSS
- Includes an image of SV Tenacious
- Responsive structure for basic viewing

## Technologies Used
- HTML
- CSS

## Project Structure
- `index.html` – Main webpage
- `style.css` – Styling for the webpage
- `Tenacious-2000.jpg` – Image used on the site
